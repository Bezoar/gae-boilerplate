JetBrains, creator of Pycharm, has given to the gae-boilerplate contributors a year-long free license.

Requirements to get you free license

In order to get this amazing gift, you have to accomplish any of these conditions:

  1. You have done more than 15 commits with an impact of more than 700 lines of code (At least one commit in the last month)
  2. You have provided more 1000 lines of code impact with at least one commit in the last month.

To see how much you have contributed visit this page

How to get this license?

Filling the contact form of the project with your email address and Github account or commenting on this issue.

About PyCharm

PyCharm, In my opinion the best Python IDE ever, is an Integrated Development Environment (IDE) used for programming in Python, It provides:

  • Code analysis
  • Graphical debugger
  • Integrated unit tester
  • Project navigation
  • Google App Engine Support
  • Project Navigation
  • User interface for Git
  • Diagrams support only in the 2.6 Early access preview
  • and many more awesome features
